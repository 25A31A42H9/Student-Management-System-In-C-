# 📚 Student Management System (C++)

## 📌 Description

This is a simple console-based **Student Management System** written in C++.
It allows users to manage student records including adding, displaying, searching, and deleting students.

---

## 🎯 Features

* Add new student
* Display all students
* Search student by ID
* Delete student
* Easy-to-use menu-driven interface

---

## 🛠️ Technologies Used

* C++
* Standard Template Library (STL - vector)

---

## ⚙️ How to Compile & Run

### Step 1: Save the file

```
student_management.cpp
```

### Step 2: Compile

```
g++ student_management.cpp -o sms
```

### Step 3: Run

Linux/Mac:

```
./sms
```

Windows:

```
sms.exe
```

---

## ▶️ Usage

1. Run the program.
2. Choose options from the menu:

   * Add student details
   * View all records
   * Search by ID
   * Delete record
3. Follow on-screen instructions.
